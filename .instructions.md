# Copilot Instructions

## file size
Please keep the file size under 100 lines to ensure maintainability and readability.

## Language
- **All code**: Use English for variable names, function names, comments, and code structure
- **User-facing UI**: Use Danish (Dansk) for UI text, labels, buttons, and user messages
- **Comments & Documentation**: Use English for all technical comments and documentation

## Project Guidelines
- React-based Backgammon tournament app for singles
- Using Supabase for backend
- Test data format includes players and games
- Follow React best practices and component structure

## Code Style
- Use functional components with React hooks
- Use inline styles or CSS modules
- Keep components focused and reusable
- Error handling with try-catch blocks

## Environment
- Development uses `npm start`
- Supabase credentials stored in `.env` file (not committed to git)
